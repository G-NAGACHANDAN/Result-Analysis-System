# Result Analysis System 📊

A comprehensive web-based application designed to automate the academic evaluation process. This system allows educators to input student marks and instantly generate deep insights through interactive visualizations and automated reporting with a single click.

## 🚀 Features

* **One-Click Analysis:** Transform raw marks into meaningful data and statistics instantly.
* **Dynamic Visualizations:** Automated generation of Bar charts, Pie charts, and Line graphs to visualize class performance.
 

* **Instant Statistics:** Real-time calculation of **Average, Highest, and Lowest** scores for every subject.
* **Multi-Format Export:** Download your detailed reports in **PDF, CSV, or Excel** formats for official documentation.
* **User-Friendly Interface:** Minimalist and intuitive design optimized for fast data entry and navigation.

## 🛠️ Tech Stack

* **Frontend:** React.js (Vite), Tailwind CSS
* **Charts:** Chart.js / Recharts
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
* **Export Libraries:** jsPDF, PapaParse (CSV)

---

## 📖 Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/yourusername/result-analysis-system.git](https://github.com/yourusername/result-analysis-system.git)
    ```

2.  **Install dependencies:**
    ```bash
    # For backend
    cd server && npm install
    
    # For frontend
    cd client && npm install
    ```

3.  **Environment Setup:**
    Create a `.env` file in the `server` directory and add your `MONGODB_URI`.

4.  **Run the application:**
    ```bash
    # Start backend
    npm run dev
    
    # Start frontend (in a new terminal)
    cd client && npm run dev
    ```

---

## 📈 Usage

1.  **Data Entry:** Navigate to the "Input Marks" section and enter student details.
2.  **Analyze:** Click the **"Generate Analysis"** button.
3.  **Visualize:** View the automatically generated charts for subject-wise and overall performance.
4.  **Export:** Select your preferred format (PDF/CSV) to save the final report.
